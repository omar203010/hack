# تطبيق التقاط الصور من الكاميرا

تطبيق ويب بسيط يسمح للمستخدمين بالتقاط صور من كاميرا الجهاز عند النقر على رابط.

## المميزات

- واجهة مستخدم جميلة وسهلة الاستخدام
- دعم اللغة العربية
- التقاط صور من الكاميرا مباشرة
- حفظ الصور تلقائياً على الخادم
- معاينة الصورة قبل الحفظ

## التثبيت

1. تثبيت المكتبات المطلوبة:
```bash
pip install -r requirements.txt
```

## التشغيل

قم بتشغيل التطبيق:
```bash
python app.py
```

ثم افتح المتصفح وانتقل إلى:
```
http://localhost:5000
```

## الاستخدام

1. افتح الرابط في المتصفح
2. انقر على "اضغط هنا لالتقاط صورة"
3. سيتم طلب الإذن للوصول إلى الكاميرا - اضغط "السماح"
4. انقر على "التقاط الصورة" لالتقاط صورة
5. يمكنك معاينة الصورة ثم حفظها أو إعادة المحاولة

## الصور المحفوظة

يتم حفظ جميع الصور في مجلد `captured_images` مع اسم فريد لكل صورة.

## ملاحظات

- التطبيق يحتاج إلى متصفح حديث يدعم WebRTC
- يجب السماح بالوصول إلى الكاميرا عند الطلب
- الصور تُحفظ بصيغة JPEG
